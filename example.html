placeholder.js

Placeholder at ::after / ::before

This is the simplest placeholder. It is displayed as text in the content of the ::after or ::before element.

Web Technologies
  • HTML

  • JavaScript

Web Technologies

Source

Below ::after is used. In order to have the placeholder in ::before you simply have to use data-placeholderjs-before.

<ul data-placeholderjs-after="The list is empty">
    <li>HTML</li>
    <li>CSS</li>
    <li>JavaScript</li>
</ul>

Placeholder Element

Sometimes the placeholder should visually replace the list. Below you can see this feature being used with MaterializeCSS colllections. Had the element not been replaced there would have been a border around the placeholder. Any HTML element can be used as a placeholder this way.
